🔍 十二宫逐宫详解

🏠 命宫 甲申 命宫

空宫，辅星左辅、天钺、铃星，杂耀天福、天德。

空宫不代表"没有"，而是"无限可能"。借对宫天机太阴为用——你的性格底色是聪明灵动机敏的，但内心有一层薄薄的敏感。左辅+天钺是双重贵人星，你天生让人想帮你。铃星是一颗暗火，表面平静、内在有主见。天福+天德是福报星——你命里有上天眷顾。

✈️ 迁移宫 戊寅

天机禄 + 太阴忌同宫，文曲+陀罗辅佐，八座+孤辰杂耀。

这是你盘中最关键的宫。天机禄代表外出、变动、学习都给你带来福气；太阴忌则让你在外面容易想太多、情绪敏感。文曲加持你的表达力，陀罗让你在变动中也有韧性。八座代表远方有贵人，孤辰则暗示你在外的某种孤独感。总结：出去有收获，但每次出去都要做心理建设。

💰 财帛宫 庚辰 身宫

巨门 + 擎羊，红鸾+台辅+天官+月德四吉杂耀。

身宫落在财帛宫，意味着你此生最在意的就是经济独立和财富自由。巨门主口舌、策划、分析——你的钱靠"说"和"想"赚来的，适合咨询、法律、媒体、教育、销售等需要表达和判断力的职业。擎羊是刀星，代表竞争激烈但也能杀出一条路。四颗吉杂耀（红鸾台辅天官月德）护财，说明你的正财运有贵人加持，尤其是在人际关系中能找到财路。

💼 官禄宫 戊子

太阳 + 文昌 + 天魁，三台+封诰+解神+咸池+天空。

这是你盘中最漂亮的宫之一！太阳在官禄代表事业光明，适合需要曝光度的职业。文昌+天魁=才华被贵人看见。三台+封诰暗示中年后有名位之喜，适合管理岗位或公众角色。解神化解困境，咸池带人缘桃花。唯一注意天空——想法多但要落地。42-51岁大限入此宫，是你事业的真正巅峰期。

💕 夫妻宫 壬午 天梁权

天梁化权 + 右弼，天厨+截路+阴煞。

天梁是荫星、孤曜，化权后你的伴侣是有权威、有原则、甚至有"长辈气质"的人——可能年龄比你大、或性格成熟稳重。右弼是助星，婚姻中有贵人扶持。但截路+阴煞暗示感情路上有隐形的障碍或误会。建议：有话直说，不要把所有敏感都藏在心里。

🧘 福德宫 丙戌

天同独坐，天喜+寡宿。

天同在福德是全书中最舒服的配置之一——你懂得享受生活、懂得在压力中找到出口。天喜带给你生活中的小确幸，寡宿则说明你享受独处时光。22-31岁的大限在此，那十年你学会了与自己和解、找到了内心的节奏。

🏥 疾厄宫 己卯

紫微科 + 贪狼 + 禄存，恩光+龙池+天寿+天使。

紫微科在疾厄是帝王星的庇护，身体素质整体不错。禄存护体，经济条件允许良好保健。贪狼需注意桃花病和应酬带来的身体负担。天寿+天使暗示需关注慢性病和女性健康。建议：定期体检，不要因为"感觉还好"就忽视。

👥 兄弟宫 癸未

廉贞 + 七杀 + 火星，天才+华盖+天月+空亡+天哭。

廉贞七杀在兄弟宫，兄弟姐妹或亲密朋友间关系较为激烈——可能有个性强烈的兄弟姐妹，或朋友圈中竞争感强。火星添急躁，天哭有空虚感。华盖+天才暗示你的朋友圈中有才华横溢之人。总之兄弟姐妹缘分偏淡，但并非无解。

👶 子女宫 辛巳

天相 + 天马，天姚+天贵+天巫+天虚。

天相在子女宫代表与晚辈关系整体和睦。天马暗示子女可能在外地发展或你们聚少离多。天巫+天贵暗示子女有灵性或学术天赋。天虚提醒不要对子女期望过高以致失望。

🏡 田宅宫 丁亥 大限32-41

武曲 + 破军，凤阁+年解。

武曲财星+破军破坏星，田宅宫的经典"拆建"组合。这十年你家宅不宁——可能是买房、搬家、装修、或家庭结构变化。但武曲是财星，变动往往伴随财务增长。凤阁暗示居住环境品位不俗，年解帮你化解房产纠纷。关键建议：房产交易务必审慎合同。

👨‍👩‍👧 父母宫 乙酉

空宫，地劫+旬空+破碎。

父母宫空宫+地劫，与父母的缘分偏淡或早年有分离经历。旬空+破碎暗示家庭中有未完成的期待或遗憾。但这不代表关系不好——只是你的人生道路更需要自己走。12-21岁大限在此，那十年你在独立中成长。

🤝 交友宫 己丑

天府 + 地空，蜚廉+天刑+天伤。

天府是库星，交友宫有此星代表朋友圈中有资源丰富的人。但地空暗示泛泛之交多、真正可靠者少。蜚廉+天刑提醒注意口舌是非和小人。52-61岁大限在此，届时你的人脉资源会迎来集中变现。
