# 三个月健身训练计划

基于现有器械 · 上下肢分化 · 复训恢复版 · 2026年6月

一个纯静态的健身训练计划 Web 应用。为零基础复训期设计，提供三种可选方案、三阶段渐进策略，以及融合传统养生功法的"五位一体"训练体系。

---

## 项目概述

专为**复训恢复期**设计（约2周荒废后重启训练），核心目标：

- 重建神经肌肉连接，唤醒肌肉
- 避免过度疲劳和严重延迟性肌肉酸痛（DOMS）
- 兼容高强度脑力工作，不影响日常状态

采用**上下肢分化（Upper/Lower Split）**策略，每块肌肉每周练2次，单日容量低、恢复快。

---

## 功能模块

### 1. 总览

训练背景与选型依据说明、现有器械清单、三种方案对比速览。

### 2. 每日必修 · 固本培元

四项每日必做的传统养生功法（不分训练日/休息日）：

| 功法 | 频率 | 核心功效 |
|------|------|----------|
| **撮谷道**（提肛缩肾） | 200次/日 | 强化盆底肌群，提升核心稳定性 |
| **叩齿** | 500下/日 | 固肾健齿，促进唾液分泌 |
| **提踵颠足** | 100下/日 | 小腿肌肉泵效应，补气升阳 |
| **八段锦** | 练后10-15分钟 | 激活副交感神经，疏通经络 |

> 力量训练练的是"外在筋骨"，撮谷道练的是"内在底座"，叩齿练的是"肾气根本"，提踵颠足练的是"气血升提"，八段锦练的是"气血流通"。**五位一体，外壮内强。**

### 3. 三个训练方案

| 方案 | 频率 | 核心思路 | 适合人群 |
|------|------|----------|----------|
| **方案一 · 四天均衡型** | 4天/周 | 胸背均衡，A/B轮换 | 时间充裕，希望快速找回全身状态 |
| **方案二 · 后侧链强化型** | 4天/周 | 拉:推 ≈ 2:1，强化背/臀/后束 | 久坐工作，体态优化需求 |
| **方案三 · 极简三天型** | 3天/周 | 超级组提效，练1休1 | 工作密集期，精力有限 |

每个方案的每个训练日都有详细的动作表，包含：顺序、器械、组次数、RPE、组间休息、动作节奏和要点。

### 4. 三个月分阶段进阶

| 阶段 | 周次 | 核心目标 | RPE |
|------|------|----------|-----|
| **阶段一 · 神经唤醒期** | 第1-3周 | 重建神经连接，找回发力感 | ≤7 |
| **阶段二 · 容量积累期** | 第4-8周 | 增加容量，刺激肌肥大 | 8 |
| **阶段三 · 强度突破期** | 第9-12周 | 加重突破力量天花板 | 8-9 |

每个阶段对应日常功课也有独立的进阶节奏。

## 使用方法

直接在浏览器中打开 `index.html`：

```
# 方式一：双击文件
双击 index.html

# 方式二：本地服务器
npx serve .
# 或
python -m http.server 8000
```

页面提供 6 个标签页切换浏览：

1. **总览** — 了解训练背景和方案对比
2. **每日必修** — 撮谷道、叩齿、提踵颠足、八段锦 完整指南
3. **方案一** ~ **方案三** — 各方案的详细训练日安排
4. **三阶段进阶** — 渐进超负荷策略与日常功课进阶节奏

## 器械前提

| 类别 | 器械 |
|------|------|
| 下肢 | 哈克深蹲机、坐姿推屈伸、坐姿髋外展、臀桥机 |
| 背部 | 引体向上架、高位下拉、直臂下拉、坐姿划船机 |
| 胸肩 | 推胸机、推肩机、蝴蝶机（正/反飞鸟） |
| 自由重量 | 哑铃（全套）、壶铃/提壶 |
| 有氧 | 跑步机、划船机 |

无杠铃并非问题——器械覆盖率极高，固定轨迹安全性更好。

---

## 训练核心规范

- **RPE 控制**：阶段一 ≤7，阶段二 8，阶段三 8-9
- **组间休息**：闭目、舌顶上颚、缓慢呼吸（4秒吸-2秒停-6秒呼），不玩手机
- **渐进超负荷触发条件**：标准动作 + 完成次数上限 + RPE ≤7 → 才可加 2-2.5kg
- **每日功课不可跳过**：宁可力量训练少做一组
